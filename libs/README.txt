Additional library folders could be mapped here for win32 builds.

freetype2 
    may be cloned, for instance, from the repository:
    git://git.sv.nongnu.org/freetype/freetype2.git
    should be compiled before poppler library build (configure + make)

poppler
    may be cloned from:
    https://anongit.freedesktop.org/git/poppler/poppler.git
    poppler\qt5\build.bat should be run before DiffPDF build
